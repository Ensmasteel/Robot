\doxysubsection{Detailed Description}
A Move\+Action will give a new order to the ghost. When start is called, the target position is given to ghost which will then go to it. (so the ghost is unlocked if necessary) By default PIDs are reset at the end of action. \begin{DoxyWarning}{Warning}
This class is abstract and cannot be instantiated. 
\end{DoxyWarning}
