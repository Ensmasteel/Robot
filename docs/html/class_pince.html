Detailed Description

The public class to construct a robot gripper. Extends the abstract class Actuator.

This class is deprecated in the 2022 edition.

Initializing function. Enables you to give the primary parameters to the pince.

Parameters
pinServo: the pin number on wich the gripper is connected.
pinDir: the pin number on wich the direction is given to the stepperMotor.
pinStep: the pin number giving the step information to the stepperMotor
pinSleep: the pin number giving the sleep information to the stepperMotor
pinM0: the pin number of the first motor of the stepperMotor.
pinM1: the second pin number of the stepperMotor.
ID: Default message ID of the stepperMotor
ferme: The initial position given in degrees.
ouvert: The final position given in degrees.
actionStep: The step of the stepperMotor.
