to execute a pure rotation, set pureRotation to True, and set posFinal to posCurrent except for _theta.

Parameters
posFinal: VectorE, target position.
deltaCurve: float, speed parameters of the trajectory.
speedRamps: float, acceleration value to reach cruising speed. [cm.-2]

pureRotation: bool, Whether if the robot only rotate or not.
backward: bool, whether the robot is going backward or not.
Returns
