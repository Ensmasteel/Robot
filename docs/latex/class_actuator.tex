\doxysubsection{Detailed Description}
Basic abstract class to define an actuator

This class defines a basic class to represent an actuator. It should be decorated with the target actuator class. The constructor is public and embeddes a name and an ID wich is \textquotesingle{}Empty\+\_\+M\textquotesingle{} as default.
