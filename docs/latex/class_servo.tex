\doxysubsubsection{\texorpdfstring{attach()}{attach()}}
{\footnotesize\ttfamily bool Servo\+::attach (\begin{DoxyParamCaption}\item[{uint8}]{pin,  }\item[{uint16}]{min\+Pulse\+Width = {\ttfamily MIN\+\_\+PULSE\+\_\+WIDTH},  }\item[{uint16}]{max\+Pulse\+Width = {\ttfamily MAX\+\_\+PULSE\+\_\+WIDTH},  }\item[{int16}]{min\+Angle = {\ttfamily MIN\+\_\+ANGLE},  }\item[{int16}]{max\+Angle = {\ttfamily MAX\+\_\+ANGLE} }\end{DoxyParamCaption})}



Associate this instance with a servomotor whose input is connected to pin. 

If this instance is already attached to a pin, it will be detached before being attached to the new pin. This function doesn\textquotesingle{}t detach any interrupt attached with the pin\textquotesingle{}s timer channel.


\begin{DoxyParams}{Parameters}
{\em pin} & Pin connected to the servo pulse wave input. This pin must be capable of PWM output.\\
\hline
{\em min\+Pulse\+Width} & Minimum pulse width to write to pin, in microseconds. This will be associated with a min\+Angle degree angle. Defaults to SERVO\+\_\+\+DEFAULT\+\_\+\+MIN\+\_\+\+PW = 544.\\
\hline
{\em max\+Pulse\+Width} & Maximum pulse width to write to pin, in microseconds. This will be associated with a max\+Angle degree angle. Defaults to SERVO\+\_\+\+DEFAULT\+\_\+\+MAX\+\_\+\+PW = 2400.\\
\hline
{\em min\+Angle} & \mbox{\hyperlink{class_target}{Target}} angle (in degrees) associated with min\+Pulse\+Width. Defaults to SERVO\+\_\+\+DEFAULT\+\_\+\+MIN\+\_\+\+ANGLE = 0.\\
\hline
{\em max\+Angle} & \mbox{\hyperlink{class_target}{Target}} angle (in degrees) associated with max\+Pulse\+Width. Defaults to SERVO\+\_\+\+DEFAULT\+\_\+\+MAX\+\_\+\+ANGLE = 180.\\
\hline
\end{DoxyParams}
@sideeffect May set pin\+Mode(pin, PWM).

\begin{DoxyReturn}{Returns}
true if successful, false when pin doesn\textquotesingle{}t support PWM. 
\end{DoxyReturn}
